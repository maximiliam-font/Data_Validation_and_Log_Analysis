## Log Processing & SQL Analysis

Short project focused on analyzing server logs using Linux CLI tools and validating data through SQL queries.

## Log Processing

- **Navigated directories and extracted patterns using grep, regex, and redirection.**
- **Filtered HTTP status codes (400/500) and organized results into structured folders.**
- **Commands used: cd, ls, grep -R, grep -E, mkdir.**

## SQL Data Analysis

- **Counted records, grouped companies, classified weather conditions, and analyzed trips by date range**
- **Queries included COUNT, GROUP BY, JOIN, CASE, and date filtering.**
